# Finans - Kişisel Finans Yönetim Uygulaması

## Hızlı Başlangıç

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npx expo start

# Android'de çalıştır
npx expo start --android

# iOS'ta çalıştır
npx expo start --ios
```

## APK Oluşturma

```bash
# EAS Build ile
eas build -p android --profile preview
```

## Özellikler

✅ Gelir/Gider Takibi  
✅ Taksit Yönetimi  
✅ Borç/Alacak Takibi  
✅ Grafiksel Analiz  
✅ Excel Dışa Aktarma  
✅ Açık/Koyu Tema
